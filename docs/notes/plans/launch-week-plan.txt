## Community Participation Strategy

### Phase 1: Listen and Learn (Days 1-7)

Don't promote yet. Just participate:

| Community | Action | Time |
|-----------|--------|------|
| Claude Code Discord | Answer questions, help others | 30 min/day |
| r/ClaudeAI | Comment on relevant posts | 20 min/day |
| r/cursor | Same as above | 20 min/day |

### Phase 2: Soft Mention (Days 8-14)

Mention Narrative only when relevant:
- "I've been working on something similar..."
- "I built a tool for this exact problem..."

### Phase 3: Direct Sharing (Days 15+)

Full promotion mode once you're a known community member.

## Pivot Triggers

**If Week 1 goes poorly:**

| Signal | Interpretation | Action |
|--------|---------------|--------|
| <20 stars | Low interest | Iterate on messaging |
| Only bug reports | Product-market fit issues | Focus on core value |
| "I don't get it" | Messaging unclear | Simplify pitch |
| "Why not just X?" | Competitive positioning | Differentiate better |
