# repo structure (high level)

Top-level folders:

- `src/` - React UI
- `src-tauri/` - Rust backend
- `docs/` - docs, screenshots, reports, and notes
- `scripts/` - local tooling
- `examples/` - sample data
- `fixtures/` - test fixtures and captured samples
- `brand/` - brand assets
- `tools/` - local repo tooling files

Top-level docs/config files:

- `README.md`, `CONTRIBUTING.md`, `SECURITY.md`, `SUPPORT.md`, `CODE_OF_CONDUCT.md`, `LICENSE`
- `package.json`, `pnpm-lock.yaml`, `vite.config.ts`, `playwright.config.ts`
- `CHANGELOG.md`, `AGENTS.md`
